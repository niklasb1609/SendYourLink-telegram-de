First step: Install Telegram on your Phone and write create a Bot by writing to Botfather
Second step: Copy the token and paste it to the token.txt file
Third step: Create a venv using: python -m venv venv
Fourth step: activate your environment using: source venv/bin/activate
Fifth step: pip install requests
Sixth step: python link.py

Navigation:
  - help for commands
  - update for getting new messages (ID is next to Name)
  - contact add <username> <id>     (Use ID from above)
  - contact rem <username>
  - contact show (shows list of all contacts)
  - send <username> <message>
  - quit (quits the program)
